    <span class="SectionTitle">Final grade calculation</span><br>
    <ul>
        <li>In order to get a final grade, <u>all projects must be presented</u>.</li>
        <li>Project <u>delays</u> result in a <u>1 point penalty per week</u>. The counting is extended all they way to the re-examination period, skipping the vacation</li>
        <li>Grade calculation formula:<br/><br/>
            <div class="Monospaced" style="margin-left: 40px; font-weight: bold">
                <span class="const">Grade</span> = <span class="num">1</span> +
                <span class="num">0.1</span>*<span class="const">RS</span> +
                <span class="num">0.2</span>*<span class="const">RV</span> +
                <span class="num">0.1</span>*<span class="const">SG</span> +
                <span class="num">0.2</span>*<span class="const">PH</span> +
                <span class="num">0.3</span>*<span class="const">TP</span>
            </div>
        </li>
    </ul>

    <span class="SectionTitle">Re-examination</span><br/>
    <ul>
        <li>Present the projects not yet presented</li>
        <li>Grades will take into account the <u>delay penalty</u> above</li>
        <li>To improve a project grade, ask the instructor, <u>ahead of time</u>, for extra work on the project in question</li>
